    й…ҚзҪ®зӨәдҫӢ¶

    жң¬ж–ҮжЎЈеҢ…еҗ«еҗ‘ Klipper Github д»“еә“пјҲдҪҚдәҺconfig directoryпјүиҙЎзҢ® Klipper й…ҚзҪ®зӨәдҫӢзҡ„жҢҮеҚ—гҖӮ

    иҜ·жіЁж„Ҹ Klipper Community Discourse server д№ҹеҸҜд»Ҙз”ЁжқҘеҜ»жүҫе’ҢеҲҶдә«й…ҚзҪ®ж–Үд»¶гҖӮ

    еҮҶеҲҷ¶

    1. йҖүжӢ©йҖӮеҪ“зҡ„й…ҚзҪ®ж–Үд»¶еҗҚеүҚзјҖпјҡ
      1. printerеүҚзјҖз”ЁдәҺдё»жөҒеҲ¶йҖ е•ҶеҮәе”®зҡ„жү“еҚ°жңәгҖӮ
      2. genericеүҚзјҖз”ЁдәҺйҖҡз”Ё3Dжү“еҚ°жңәдё»жқҝгҖӮ
      3. kitзҡ„еүҚзјҖз”ЁдәҺжҢүз…§е…¬ејҖи§„иҢғз»„иЈ…зҡ„3Dжү“еҚ°жңәпјҲдҫӢеҰӮVoron V2.4пјүгҖӮиҝҷдәӣ "еҘ—д»¶ "жү“еҚ°жңәйҖҡеёёдёҺжҷ®йҖҡзҡ„жү“еҚ°жңәдёҚеҗҢеңЁе®ғд»¬йҖҡеёёдёҚиў«еҲ¶йҖ е•Ҷй”Җе”®гҖӮ
      4. sampleеүҚзјҖз”ЁдәҺеҸҜд»Ҙиў«еӨҚеҲ¶еҲ°дё»й…ҚзҪ®ж–Үд»¶дёӯзҡ„й…ҚзҪ® "зүҮж®ө"гҖӮ
      5. exampleеүҚзјҖжҳҜз”ЁжқҘжҸҸиҝ°жү“еҚ°жңәиҝҗеҠЁеӯҰгҖӮиҝҷз§Қзұ»еһӢзҡ„й…ҚзҪ®йҖҡеёёеҸӘдёҺж–°зұ»еһӢзҡ„жү“еҚ°жңәиҝҗеҠЁеӯҰзҡ„д»Јз ҒдёҖиө·ж·»еҠ гҖӮ
    2. жүҖжңүзҡ„й…ҚзҪ®ж–Үд»¶еҝ…йЎ»д»Ҙ.cfgеҗҺзјҖз»“е°ҫгҖӮprinterй…ҚзҪ®ж–Үд»¶еҝ…йЎ»д»Ҙ.cfgз»“е°ҫпјҲдҫӢеҰӮпјҢ2019.cfgпјүгҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢе№ҙд»ҪжҳҜеҜ№еә”жү“еҚ°жңәзҡ„еӨ§иҮҙеҸ‘е”®е№ҙд»ҪгҖӮ
    3. дёҚиҰҒеңЁй…ҚзҪ®ж–Үд»¶еҗҚдёӯдҪҝз”Ёз©әж јжҲ–зү№ж®Ҡеӯ—з¬ҰгҖӮж–Үд»¶еҗҚеә”иҜҘеҸӘеҢ…еҗ«еӯ—з¬ҰA-Z, a-z, 0-9, -, е’Ң.гҖӮ
    4. printer, generic, е’Ң kit зӨәдҫӢй…ҚзҪ®ж–Үд»¶еҝ…йЎ»дҝқиҜҒ Klipper иғҪеӨҹжӯЈеёёеҗҜеҠЁиҖҢдёҚеҮәй”ҷгҖӮиҝҷдәӣй…ҚзҪ®ж–Үд»¶еә”иҜҘиў«ж·»еҠ еҲ° test/klippy/printers.test еӣһеҪ’жөӢиҜ•з”ЁдҫӢдёӯгҖӮе°Ҷж–°зҡ„й…ҚзҪ®ж–Үд»¶ж·»еҠ еҲ°иҜҘжөӢиҜ•з”ЁдҫӢзҡ„йҖӮеҪ“йғЁеҲҶпјҢе№¶жҢүиҜҘйғЁеҲҶзҡ„еӯ—жҜҚйЎәеәҸжҺ’еҲ—гҖӮ
    5. иҜҘй…ҚзҪ®зӨәдҫӢеә”иҜҘжҳҜжү“еҚ°жңәзҡ„ "stock "й…ҚзҪ®гҖӮ(еңЁklipperзҡ„д»“еә“дёӯжңүеӨӘеӨҡе®ҡеҲ¶зҡ„й…ҚзҪ®гҖӮ)еҗҢж ·ең°пјҢжҲ‘д»¬еҸӘдёәе…·жңүдё»жөҒжөҒиЎҢжҖ§зҡ„жү“еҚ°жңәгҖҒеҘ—д»¶е’Ңжқҝеӯҗж·»еҠ й…ҚзҪ®ж–Үд»¶зҡ„дҫӢеӯҗпјҲиҮіе°‘еә”иҜҘжңү100дёӘжӯЈеңЁдҪҝз”ЁдёӯпјүгҖӮиҖғиҷ‘дҪҝз”ЁKlipper Community Discourse serverиҝӣиЎҢе…¶д»–й…ҚзҪ®гҖӮ
    6. Only specify those devices present on the given printer or board. Do not specify settings specific to your particular setup.
      1. For generic config files, only those devices on the mainboard should be described. For example, it would not make sense to add a display config section to a "generic" config as there is no way to know if the board will be attached to that type of display. If the board has a specific hardware port to facilitate an optional peripheral (eg, a bltouch port) then one can add a "commented out" config section for the given device.
      2. дёҚиҰҒеңЁй…ҚзҪ®зӨәдҫӢдёӯжҢҮе®ҡpressure_advanceпјҢеӣ дёәиҜҘеҖјжҳҜй’ҲеҜ№иҖ—жқҗзҡ„пјҢиҖҢдёҚжҳҜжү“еҚ°жңәзЎ¬д»¶гҖӮеҗҢж ·пјҢдёҚиҰҒжҢҮе®ҡmax_extrude_only_velocityжҲ–max_extrude_only_accelи®ҫзҪ®гҖӮ
      3. дёҚиҰҒжҢҮе®ҡдёҖдёӘеҢ…еҗ«дё»жңәи·Ҝеҫ„жҲ–дё»жңәзЎ¬д»¶зҡ„й…ҚзҪ®йғЁеҲҶгҖӮдҫӢеҰӮпјҢдёҚиҰҒжҢҮе®ҡ[virtual_sdcard]жҲ–[temperature_host]й…ҚзҪ®йғЁеҲҶгҖӮ
      4. еҸӘдёәеҲ©з”Ёзү№е®ҡжү“еҚ°жңәеҠҹиғҪпјҢжҲ–зү№е®ҡжү“еҚ°жңәй…ҚзҪ®зҡ„еҲҮзүҮиҪҜд»¶йҖҡеёёеҸ‘еҮәзҡ„Gд»Јз Ғе®ҡд№үе®ҸгҖӮ
    7. Where possible, it is best to use the same wording, phrasing, indentation, and section ordering as the existing config files.
      1. The top of each config file should list the type of micro-controller the user should select during "make menuconfig". It should also have a reference to "docs/Config_Reference.md".
      2. дёҚиҰҒе°Ҷеӯ—ж®өж–ҮжЎЈеӨҚеҲ¶еҲ°зӨәдҫӢй…ҚзҪ®ж–Үд»¶дёӯгҖӮпјҲиҝҷж ·еҒҡдјҡйҖ жҲҗз»ҙжҠӨж–№йқўзҡ„иҙҹжӢ…пјҢеӣ дёәеҜ№ж–ҮжЎЈзҡ„жӣҙж–°йңҖиҰҒеңЁеҫҲеӨҡең°ж–№иҝӣиЎҢдҝ®ж”№гҖӮпјү
      3. й…ҚзҪ®ж–Үд»¶зҡ„дҫӢеӯҗдёҚеә”еҢ…еҗ« "SAVE_CONFIG "йғЁеҲҶгҖӮеҰӮжһңжңүеҝ…иҰҒпјҢжҠҠSAVE_CONFIGйғЁеҲҶзҡ„зӣёе…іеӯ—ж®өеӨҚеҲ¶еҲ°дё»й…ҚзҪ®еҢәзҡ„йҖӮеҪ“йғЁеҲҶгҖӮ
      4. дҪҝз”Ёfield: valueзҡ„иҜӯжі•пјҢиҖҢдёҚиҰҒдҪҝз”Ёfield=valueгҖӮ
      5. еҪ“и®ҫе®ҡдёҖдёӘжҢӨеҮәжңәзҡ„rotation_distanceж—¶пјҢеҰӮжһңжҢӨеҮәжңәжңүдёҖдёӘйҪҝиҪ®жңәжһ„пјҢжңҖеҘҪжҢҮе®ҡgear_ratioгҖӮжҲ‘д»¬еёҢжңӣзӨәдҫӢй…ҚзҪ®дёӯзҡ„ж—ӢиҪ¬и·қзҰ»дёҺжҢӨеҮәиҪ®зҡ„е‘Ёй•ҝзӣёе…і--е®ғйҖҡеёёеңЁ20еҲ°35жҜ«зұід№Ӣй—ҙгҖӮеҪ“жҢҮе®ҡgear_ratioж—¶пјҢжңҖеҘҪжҢҮе®ҡжңәжһ„дёҠзҡ„е®һйҷ…йҪҝиҪ®йҪҝж•°пјҲдҫӢеҰӮпјҢдјҳе…ҲйҖүжӢ©gear_ratio: 80:20иҖҢдёҚжҳҜgear_ratio: 4:1пјүгҖӮеҸӮи§Ғж—ӢиҪ¬и·қзҰ»ж–ҮжЎЈд»ҘиҺ·еҫ—жӣҙеӨҡдҝЎжҒҜгҖӮ
      6. йҒҝе…Қе®ҡд№үйӮЈдәӣиў«и®ҫзҪ®дёәй»ҳи®ӨеҖјзҡ„еӯ—ж®өеҖјгҖӮдҫӢеҰӮпјҢдёҚеә”иҜҘжҢҮе®ҡmin_extrude_temp: 170пјҢеӣ дёәиҝҷе·Із»ҸжҳҜй»ҳи®ӨеҖјгҖӮ
      7. еңЁеҸҜиғҪзҡ„жғ…еҶөдёӢпјҢиЎҢж•°дёҚеә”и¶…иҝҮ80еҲ—гҖӮ
      8. йҒҝе…ҚеңЁй…ҚзҪ®ж–Үд»¶дёӯж·»еҠ еҪ’еұһжҲ–дҝ®и®ўдҝЎжҒҜгҖӮдҫӢеҰӮпјҢйҒҝе…Қж·»еҠ зұ»дјј "жӯӨж–Үд»¶з”ұ......еҲӣе»ә "зҡ„иЎҢгҖӮпјүе°ҶеҪ’еұһе’Ңдҝ®ж”№еҺҶеҸІж”ҫеңЁgitжҸҗдәӨдҝЎжҒҜдёӯгҖӮ
    8. дёҚиҰҒеңЁзӨәдҫӢй…ҚзҪ®ж–Үд»¶дёӯдҪҝз”Ёд»»дҪ•е·Іеәҹејғзҡ„еҠҹиғҪгҖӮ
    9. дёҚиҰҒеңЁзӨәдҫӢй…ҚзҪ®ж–Үд»¶дёӯзҰҒз”Ёй»ҳи®Өе®үе…Ёзі»з»ҹгҖӮдҫӢеҰӮпјҢдёҖдёӘй…ҚзҪ®дёҚеә”иҜҘжҢҮе®ҡдёҖдёӘиҮӘе®ҡд№үзҡ„ max_extrude_cross_sectionгҖӮдёҚиҰҒеҗҜз”Ёи°ғиҜ•еҠҹиғҪгҖӮдҫӢеҰӮпјҢдёҚеә”иҜҘжңүдёҖдёӘ force_move й…ҚзҪ®йғЁеҲҶгҖӮ
    10. Klipperж”ҜжҢҒзҡ„жүҖжңүе·ІзҹҘжқҝеӯҗйғҪеҸҜд»ҘдҪҝз”Ёй»ҳи®Өзҡ„дёІиЎҢжіўзү№зҺҮ250000гҖӮдёҚжҺЁиҚҗдҪҝз”ЁдёҚеҗҢзҡ„жіўзү№зҺҮеңЁзӨәдҫӢж–Үд»¶дёӯгҖӮ
